Products.pruebaFahce Installation
---------------------------------

To install Products.pruebaFahce using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``Products.pruebaFahce`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        Products.pruebaFahce
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
